Display image from url.

== Description ==

Creates an shortcode to display an image from any valid URL.

i.e.
[image url="http://www.papeldeparede.etc.br/fotos/wp-content/uploads/praia.jpg" style="border: none;" class="myClass"]

== Installation ==

Just copy the plugin files to the plugin folder.
